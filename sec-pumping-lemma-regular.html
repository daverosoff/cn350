    Section 2.5 Using the Pumping Lemma

    Subsection 2.5.1 Class activities

    Suppose that \(L\) is a language we wish to prove is not regular, using the Pumping Lemma. The proof begins like this: “If \(L\) is regular, then the Pumping Lemma says it must have a pumping constant \(p\)”. The Pumping Lemma says that every word of length at least \(p\) can be split into three parts, \(xyz\text{,}\) where \(|xy| \leq p\) and \(|y| \geq 1\text{,}\) such that \(xy^kz \in L\) for all \(k \geq 0\text{.}\) We sometimes call \(y\) a pumpable substring or even just a pump. The proof generally proceeds by finding a word \(w\) that is in \(L\) and that has no pumpable substring. This is a contradiction of the Pumping Lemma, and shows our assumption (that \(L\) was regular) must be false.

    Activity 2.5.1.

    Which of the following are true?
    1. We get to choose a value for \(p\text{.}\)
    2. We get to choose a candidate word \(w\) of length at least \(p\text{.}\)
    3. We get to choose the parts (that is, \(x\text{,}\) \(y\text{,}\) and \(z\)) in the decomposition \(w = xyz\text{.}\)
    4. We get to choose a value for \(k\text{.}\)
    Answer.
    Only Item B and Item D are true.
    Solution.
    The pumping lemma does not relate the value of \(p\) to the language \(L\text{.}\) It only states that \(p\) exists and has the specified properties. Given the length \(p\text{,}\) we can choose any convenient word \(w\) as long as \(\abs{w} \geq p\text{.}\) However, we have to examine all the decompositions, so we don’t get to choose \(x\text{,}\) \(y\text{,}\) or \(z\text{.}\) We only have to show that (for every choice of \(x\text{,}\) \(y\text{,}\) \(z\)) there is some value of \(k\) such that \(xy^kz \notin L\text{,}\) so we get to choose \(k\text{.}\)

    Activity 2.5.2.

    What is a good choice of \(w\) for proving that each of the following languages is not regular?
    (a)
    \(\setof{a^nb^n}{n \geq 0}\)
    (b)
    \(\setof{a^{2n}b^n}{n \geq 0}\)
    (c)
    \(\setof{x \in \aster{\{a, b\}}}{x \text{ has equally many $a$ and $b$}}\)

    Activity 2.5.3.

    A string of left brackets ( and right brackets ) is balanced if every left bracket has a matching right bracket. For example, (), (()), and (())() are balanced, but )(, ())), and ()((()) are not. What is a good choice of \(w\) for proving that \(\setof{x \in \aster{\{\mathtt{(}, \mathtt{)}\}}}{x \text{ is balanced}}\) is not regular?
    The balanced bracket languages play an important role in the structure theory of the Chomsky hierarchy, as we’ll see later on. They are also connected to an interesting numerical sequence, the Catalan numbers. You can learn more about the Catalan numbers at Wikipedia. There are many diagrams there which I expect you will find entertaining.

    Activity 2.5.4.

    Here are the first 8 languages proved non-regular in this video 4 .
    1. \(\displaystyle \{ 0^n 1^n \}\)
    2. \(\displaystyle \{ \text{ equal $0$s and $1$s } \}\)
    3. \(\displaystyle \{ \text{ more $1$s than $0$s } \}\)
    4. \(\displaystyle \{ 0^{2n} 1^n \}\)
    5. \(\displaystyle \{ 0^n 1 0^n \}\)
    6. \(\displaystyle \{ 0^m 1^n 0^{m+n} \}\)
    7. \(\displaystyle \{ 0^m 1^n 0^{mn} \}\)
    8. \(\displaystyle \setof{0^n 1^m}{n \neq m}\)
    Watch the video segment for the language of your choice. Then try to reproduce the proof without looking back. It’s annoying to write proofs during class, but try. Work in groups of 4 or more (so I can be sure to see everyone’s proof).
    The last exercise in the Activity 2.5.4 needs a special trick. The reason is that even filling \(xy\) with 0s doesn’t help. One trick that works is the so-called  “\(p\)-factorial trick”.
    We choose \(w = 0^p1^{p+p!}\) (still looking at the language \(L = \setof{0^n 1^m}{n \neq m}\)). Then as usual, \(x\) and \(y\) are all 0s. But notice that we must have \(1 \leq y \leq p\text{.}\) It follows that \(\abs{y}\) divides \(p!\text{.}\) So if we choose \(k = p!/y\text{,}\) then \(ky = p!\text{.}\) Then \(xy^kz = 0^{p+p!} 1^{p+p!} \notin L\text{.}\) This completes the proof that \(L\) is not regular.
    The \(p!\) trick can be helpful when, as in the previous example, we find it harder to get out of the language than to stay in.

    Example 2.5.2.

    Use the pumping lemma to show that \(L = \setof{0^n}{\text{$n$ is prime}}\) is not regular.
    Here, we have to be quite sneaky. The first problem is that the pumping constant probably isn’t prime, but even if it is we have no way of knowing that. So we have to rely on the fact that the set of primes is unbounded: there is a prime greater than any given number. So we can choose \(q\) to be the smallest prime greater than \(p\) and then choose \(w = 0^{q+2}\text{.}\) The reason for the \(2\) will be clear soon.
    Now \(xy\) is all zeroes, since \(w\) is all zeroes. So \(y\) is all zeroes. We need a value of \(k\) such that \(\abs{xy^kz} = \abs{xz} + k \abs{y}\) is not prime. We cleverly let \(k = \abs{xz}\text{.}\) Then
    \begin{align*} \abs{xy^kz} \amp = \abs{xz} + k \abs{y}\\ \amp = \abs{xz} + \abs{xz} \abs{y}\\ \amp = \abs{xz} (1 + \abs{y})\text{.} \end{align*}
    Now, because \(\abs{y} \geq 1\text{,}\) we have \(1 + \abs{y} \geq 2\text{.}\) If \(\abs{xz} \geq 2\) , the proof is complete, because we have a factorization of \(\abs{xy^kz}\) into two numbers greater than 1. So we just have to show that \(\abs{xz} \geq 2\text{.}\) This is where the \(2\) in \(w = 0^{q+2}\) comes in. We have \(\abs{xyz} \geq p + 2\text{,}\) which is to say
    \begin{equation*} \abs{xy} + \abs{z} \geq p + 2\text{.} \end{equation*}
    We have to be careful with inequalities. But if we subtract \(\abs{xy}\) from the left, we know we have reduced the left by at most \(p\) (possibly less), so we may subtract \(p\) from the right and the inequality will still hold. So we have
    \begin{equation*} \abs{z} \geq 2\text{,} \end{equation*}
    as required.
