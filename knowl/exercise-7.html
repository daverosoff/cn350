Each of the languages below is the concatenation of two simpler languages. The concatenation of languages \(B\) and \(C\) is the language \(BC = \{xy : x \in B \text{ and } y \in C\}\text{.}\) For each language, give a single \(\varepsilon\)-NFA that accepts it. The language is always \(\Sigma = \{a, b\}\text{.}\)
  1. Zero or more \(a\) followed by zero or more \(b\)
  2. A string of odd length followed by a string ending in \(aab\)
  3. Any string except a single \(a\) or \(b\)
