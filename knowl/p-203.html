Let \(q\) be a state in an \(\varepsilon\)-NFA. The \(\varepsilon\)-closure of \(q\text{,}\) written \(\textrm{eclose}(q)\text{,}\) is a set of states in the NFA defined inductively as follows.
  1. The state \(q\) is in \(\textrm{eclose}(q)\text{.}\)
  2. If state \(p\) is in \(\textrm{eclose}(q)\text{,}\) and there is an \(\varepsilon\)-transition from \(p\) to \(r\text{,}\) then \(r\) is in \(\textrm{eclose}(q)\text{.}\)
