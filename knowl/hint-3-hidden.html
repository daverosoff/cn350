The base case is \(n = 0\) just like before. The inductive step is to prove: if \(6\) divides \(n^3 - n\text{,}\) it also divides \((n+1)^3 - (n+1)\text{.}\)
Here are some helpful facts that you can use without proof. In a math class, we’d prove these as exercises as well, but here I’m hoping to provide enough math hints that you can focus more on the logical structure of the argument and less on the arithmetic.
  1. If \(3\) divides \(\ell\) and \(2\) divides \(\ell\text{,}\) then \(6\) divides \(\ell\text{.}\)
  2. “\(a\) is even” is the same statement as “\(2\) divides \(a\)”.
  3. If \(a\) divides \(b\text{,}\) then \(a\) divides \(kb\) for all integers \(k\text{.}\)
  4. If \(a\) divides \(b\) and \(a\) divides \(c\text{,}\) then \(a\) divides \(b \pm c\text{.}\)
