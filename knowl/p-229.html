The definition is inductive. Let the alphabet of symbols be some set \(\Sigma\text{.}\) Then
  1. The constants \(\varepsilon\) and \(\varnothing\) are regular expressions. We define \(L(\varepsilon) = \{\varepsilon\}\) and \(L(\varnothing) = \varnothing\text{.}\)
  2. If \(a \in \Sigma\text{,}\) then \(a\) is a regular expression and \(L(a) = \{a\}\text{.}\)
  3. A variable, usually capitalized like \(B\text{,}\) is a regular expression. Such a variable represents a whole language.
  4. If \(E\) and \(F\) are regular expressions, then so is their union \(E + F\text{.}\) The language it represents is \(L(E) + L(F)\text{.}\)
  5. If \(E\) and \(F\) are regular expressions, then so is their concatenation \(E F\text{.}\) The language it represents is \(L(E) L(F)\text{.}\)
  6. If \(E\) is a regular expression, then so is its star \(\aster{E}\text{.}\) The language it represents is \(\aster{L(E)}\text{.}\)
  7. Finally, if \(E\) is a regular expression, then so is its parenthesization \((E)\text{.}\) The language it represents is \(L(E)\text{.}\)
