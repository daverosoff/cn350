Definition 2.3.15.

Let \(M\) be an \(\varepsilon\)-NFA. Then the extended transition function of \(M\text{,}\) written \(\hat{\delta}\text{,}\) is a function \(\hat{\delta} \colon Q \times \aster{\Sigma} \to 2^Q\) defined as follows.
  1. \(\hat{\delta}(q, \varepsilon) = \textrm{eclose}(q)\) for each state \(q\text{.}\)
  2. \(\hat{\delta}(q, xa)\) is defined as follows. First let \(p_1, \ldots, p_k\) be the members of the set \(\hat{\delta}(q, x)\text{.}\) Then let \(r_1, \ldots, r_m\) be the members of the set
    \begin{equation*} \bigcup_{i=1}^k \hat{\delta}(p_i, a)\text{.} \end{equation*}
    Then we define \(\hat{\delta}(q, xa)\) to be the set
    \begin{equation*} \textrm{eclose}(\{r_1, \ldots, r_m\})\text{.} \end{equation*}
