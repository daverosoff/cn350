Task 1.2.4.a.
Consider a rectangular \(2 \times n\) array of squares, and the problem of determining the number of ways it may be tiled by \(n\) dominoes. Each domino must cover exactly 2 adjacent squares, without overlaps. So, one way is to use all parallel dominoes. If \(n\) happens to be even, all horizontal dominoes will work as well.
Of course, other tilings are also possible.
You can probably draw all 13 tilings of the \(2 \times 6\) grid quite easily. It turns out that 13 is also the value of \(F_7\text{,}\) the 7th Fibonacci number. This isn’t a coincidence, as you’re asked to show below.
Using the recurrence relation above and the induction framework, show that there are \(F_{n+1}\) such tilings of the \(2 \times n\) array by dominoes, for all \(n \geq 0\text{.}\)
Note. For \(n = 0\text{,}\) we say that there is one way to tile an empty array with no dominoes. There would be zero ways to do it with more dominoes, and zero ways to tile a nonempty array with no dominoes.
