Task 3.3.1.b.
The class NFA. This class should have the following methods. My remarks above about the parameters apply here as well.
  1. a constructor that returns a usable NFA object
  2. a __repr__() method that dumps the current formal specification of the NFA to the output
  3. a helper method do_delta() that simulates the (non-extended) transition function \(\delta\text{.}\) This method should handle missing transitions gracefully, since it is for an NFA.
  4. a run() method that simulates the action of the extended transition function \(\hat{\delta}\) (see DefinitionÂ 2.3.2). This method should call the do_delta() method. It should return a bool whose value indicates acceptance or rejection.
