Activity 2.4.7.

Suppose you are given an NFA \(N\) having
  1. a single start state with no incoming transitions
  2. a single accept state with no outgoing transitions.
Explain how you would use \(N\) to build a new NFA \(N\) satisfying \(L(M) = \aster{L(N)}\text{.}\) \(M\) should also satisfy conditions 1 and 2 above. You are allowed to use \(\varepsilon\)-transitions and to respecify the start and accepting states.
