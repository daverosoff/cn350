The star operator represents the set of words formed from zero or more concatenations of the language to which it is applied. For example, \(\aster{L}\) denotes the set of strings obtaining by pasting zero or more strings from \(L\) together. Another way to say it is:
\begin{equation*} \aster{L} = L^0 + L^1 + L^2 + \cdots \end{equation*}
where the powers represent self-concatenation.
