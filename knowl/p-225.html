The concatenation \(L_1 L_2\) of two languages consists of the words \(w = xy\) where \(x\) matches \(L_1\) and \(y\) matches \(L_2\text{.}\) Notice that only one such decomposition is required to exist: if itâ€™s possible to decompose \(w\) this way, it is in \(L_1 L_2\text{.}\) As you see, the concatenation operator is usually not represented by an explicit symbol, but by juxtaposition. We frequently use powers to stand for self-concatenation. So, \(L^2\) would be the same as \(L L\text{,}\) and \(L^3\) would be the same as \(L L L = L^2 L = L L^2\text{,}\) and so on. Using this convention, we have \(L^0 = \{\epsilon\}\) and \(L^1 = L\) for all languages \(L\text{.}\)
