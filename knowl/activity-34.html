Activity 2.4.6.

Suppose you are given NFA \(M_1\) and \(M_2\) each having
  1. a single start state with no incoming transitions
  2. a single accept state with no outgoing transitions.
Explain how you would combine these into a new NFA \(M\) satisfying \(L(M) = L(M_1) L(M_2)\text{.}\) \(M\) should also satisfy conditions 1 and 2 above. You are allowed to use \(\varepsilon\)-transitions and to respecify the start and accepting states.
