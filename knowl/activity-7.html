Activity 1.1.7.

We say a set \(A\) is finite if there is a nonnegative integer \(n\) such that the proposition
\begin{equation} A \text{ has $n$ elements}\tag{1.1.1} \end{equation}
is true. If \(A\) is a finite set, then \(\abs{A}\) denotes the size of \(A\text{,}\) the number of its elements.
A one-to-one correspondence or bijection is a function that can be reversed. Given a function \(f \colon A \to B\text{,}\) you can tell that \(f\) is a bijection if it pairs each “input” (argument) to a unique output (value).
(a)
Let \(A\) be a finite set. How many functions are there with domain \(A\) and codomain \(\mathbf{\overline{2}} = \{0,1\}\text{?}\)
