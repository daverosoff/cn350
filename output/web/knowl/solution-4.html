Solution 1.

\(\text{Billy and Bob are applying for the same job, but only one can succeed.}\) \(\text{p: Billy gets the job}\) \(\text{q: Bob gets the job}\) First, translate the statement into formal logic. In this statement, \(p\) can be true, xor \(q\) can be true, or neither can be true. The key here is that while either one can succeed, there is no guarantee of success. The expression is then equal to \(\mathbin{\sim} (p\wedge q)\) Now, negate the statement. \(\mathbin{\sim} ( \mathbin{\sim} (p\wedge q))\) \(\equiv \mathbin{\sim} \mathbin{\sim} (p\wedge q)\) \(\equiv p\wedge q\) The answer is then \(p\wedge q\) As an extra exercise, what would this statement translate to in English?
