Define a function
\begin{equation*} \mathrm{ev}_1 \colon \{ f : f \text{ is a function $A \to \mathbf{\overline{2}}$} \} \to 2^A \end{equation*}
by the equation \(\mathrm{ev}_1(f) = \{ x \in A : f(x) = 1 \}\text{.}\) We will show that \(\mathrm{ev}_1\) is a bijection. Bijections are functions with two properties:
  1. If \(s \ne t\) then \(G(s) \ne G(t)\) (no overlap/collision; at most one inbound arrow for each codomain element)
  2. For each \(y\) in the codomain, there is \(x\) in the domain such that \(G(x) = y\) (every codomain element “covered”; at least one inbound arrow for each codomain element)
Suppose that \(f\) and \(g\) are distinct functions \(A \to \mathbf{\overline{2}}\text{.}\) We need to show that \(\mathrm{ev}_1(f) \ne \mathrm{ev}_1(g)\text{.}\) The first is the subset of \(A\) on which \(f\) takes the value \(1\text{.}\) The second is the same set, but for \(g\text{.}\) Since \(f \ne g\text{,}\) there is some \(x \in A\) where \(f(x) \ne g(x)\text{.}\) Since the only possible values of these functions are \(0\) and \(1\text{,}\) it follows that exactly one of \(f(x)\) and \(g(x)\) is equal to \(1\text{.}\) Hence \(x\) is a member of \(\mathrm{ev}_1(f)\) or \(\mathrm{ev}_1(g)\text{,}\) but not both. This shows that \(\mathrm{ev}_1\) has the first property of a bijection.
To obtain the second, let us choose an arbitrary element \(S \in 2^A\text{.}\) This \(S\) is just a subset of \(A\text{.}\) We must show that there exists a function \(h\) such that \(\mathrm{ev}_1(h) = S\text{.}\) We may of course define
\begin{equation*} h(x) = \begin{cases} 0 \amp \text{if $x \notin S$} \\ 1 \amp \text{if $x \in S$} \\ \end{cases} \end{equation*}
and then we see at once that \(h\) has the desired property. We have now shown that \(\mathrm{ev}_1\) has both properties of a bijection.
