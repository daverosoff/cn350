    Section 2.3 Nondeterministic finite automata

    In this section we have learned that NFAs are much like DFAs, but with more flexibility about the transitions. Recall that DFA are required to have a transition function with the signature \(Q \times \Sigma \to Q\text{.}\) The definition of function means that there is thus exactly one outbound transition for each symbol and state.
    With an NFA, this restriction is removed. A given state and symbol may yield 0, 1, or many outbound transitions. When there are 0, the computation can go no further, but when there are many, the automaton follows them all to their ends, by the mechanism of the redesigned transition function.
    The transition function for NFA has a different signature. It is a function \(\delta \colon Q \times \Sigma \to 2^Q\text{,}\) where as before the exponentiated set is a power set. The meaning is simple: the value of \(\delta(q, a)\) is the set of states that can be reached from \(q\) by following edges labeled \(a\). That‚Äôs it!

    Activity 2.3.1.

    Consider the following NFA:
    Evidently it accepts all words ending in \(01\text{.}\) Give its formal components \(Q\text{,}\) \(\Sigma\text{,}\) \(q_0\text{,}\) \(F\text{,}\) and \(\delta\text{.}\)
    Answer.
    The first four are
    \begin{align*} Q \amp = \{ q_0, q_1, q_2 \}\\ \Sigma \amp = \{ 0, 1 \}\\ s \amp = q_0\\ F \amp = \{ q_2 \}\text{,} \end{align*}
    and \(\delta\) is given by the table below.
    0 1
    \(\to q_0\) \(\{ q_0, q_1 \}\) \(\{ q_0 \}\)
    \(q_1\) \(\varnothing\) \(\{ q_2 \}\)
    \(\ast q_2\) \(\varnothing\) \(\varnothing\)
    Notice how the table looks quite similar to those of Section¬†2.2. The only difference is that the function values are sets of states rather than individual states.
    We defined the language \(L(M)\) of a DFA \(M\) using the extended transition function (defined in Definition¬†2.2.2) as follows:
    \begin{equation*} L(M) = \{ x \in \aster{\Sigma} : \hat{\delta}(s, x) \in F \}\text{,} \end{equation*}
    where \(s\) is the start state of \(M\) and \(F\) is the set of accepting states. In order to define such a thing for NFA, we will probably need such an extended version of its transition function. This is also provided for us.
    We defined the extended transition function \(\hat{\delta}\) by
    \begin{equation*} \hat{\delta}(q, w) = \begin{cases} \{q\} \amp \text{if $w = \varepsilon$} \\ \displaystyle \bigcup_{i=1}^k \delta(p_i, a) \amp \text{if $w = xa$ with $a \in \Sigma$}. \\ \end{cases} \end{equation*}
    To interpret this we must know what the states \(p_1\text{,}\) \(p_2, \ldots p_k\) are. These are all the states we can reach by following the symbols in the prefix \(x\) from the state \(q\text{.}\) Remember, this is a recursive/inductive definition, so we learn how to do an \(n\)-symbol word from how to do an \((n-1)\)-symbol one. The second case in the definition says, follow the \(a\)-transitions from every state \(p_i\text{,}\) \(i = 1\text{,}\) \(2, \ldots, k\text{,}\) and the states you get in this way are all members of \(\hat{\delta}(q, w)\text{.}\)

    Example 2.3.2.

    Let us compute \(\hat{\delta}(q_0, {\tt 00101})\) using the inductive definition. It is not so easy to do the computation in the intuitive way, starting with the long string and working down to the empty string. The reason is simple: we don‚Äôt know the \(p_i\text{!}\) But, we know from experience by now that we will end up evaluating \(\hat{\delta}(q_0, x)\) for every prefix \(x\) of \({\tt 00101}\text{.}\) It will make our job simpler if we start with \(\varepsilon\) and work up from there.
    1. \(\hat{\delta}(q_0, \varepsilon) = \{q_0\}\text{,}\) by definition. Since we can only reach the single state \(q_0\) ‚Äúso far‚Äù, it is the only \(p_i\) for the next step.
    2. \(\hat{\delta}(q_0, 0) = \delta(q_0, 0) = \{ q_0, q_1 \}\text{.}\) Here there is only one \(p_i\)‚Äî\(q_0\) from above‚Äîand only one symbol, so the union is just the set \(\{q_0, q_1\}\text{.}\) These are the \(p_i\) for the next step.
    3. \(\hat{\delta}(q_0, 00) = \delta(q_0, 0) \cup \delta(q_1, 0) = \{ q_0, q_1 \} \cup \varnothing = \{ q_0, q_1 \}\text{.}\) Here there are two states \(p_1\) and \(p_2\text{,}\) so two things in the union, but one of them is empty and we get nothing new yet.
    4. \(\hat{\delta}(q_0, 001) = \delta(q_0, 1) \cup \delta(q_1, 1) = \{ q_0 \} \cup \{ q_2 \} = \{ q_0 , q_2 \}.\) This time, both things in the union are nonempty.
    5. \(\displaystyle \hat{\delta}(q_0, 0010) = \delta(q_0, 0) \cup \delta(q_2, 0) = \{ q_0, q_1 \} \cup \varnothing = \{q_0, q_1\}.\)
    6. \(\displaystyle \hat{\delta}(q_0, 00101) = \delta(q_0, 1) \cup \delta(q_1, 1) = \{ q_0 \} \cup \{ q_2 \} = \{ q_0, q_2 \}.\)
    Later we will see that this means the word \({\tt 00101}\) is accepted by this machine, because \(\hat{\delta}(q_0, 00101)\) contains a state of \(F\text{.}\)
    Equipped with the extended transition function, we can define the language of an NFA.

    Definition 2.3.3.

    If \(M\) is our NFA, we define
    \begin{equation*} L(M) = \{ w \in \aster{\Sigma}: \hat{\delta}(q_0, w) \cap F \ne \varnothing \}\text{.} \end{equation*}

    Example 2.3.4.

    Let us prove that \(M\text{,}\) the NFA of Figure¬†2.3.1, accepts the language \(L = \{ w : w = y01 \}\text{.}\) That is, we wish to prove that
    \begin{equation*} L(M) = L, \end{equation*}
    where
    \begin{equation*} L(M) = \{ w \in \{0,1\}^{\ast} : q_2 \in \hat{\delta}(q_0, w) \}\text{.} \end{equation*}
    Make sure you see why the defining property \(\hat{\delta}(q_0, w) \cap F \ne \varnothing\) from Definition¬†2.3.3 becomes \(q_2 \in \hat{\delta}(q_0, w)\) with our choice of \(M\text{.}\)
    We will use induction, but of the ‚Äúmutual‚Äù variety. Here are the three statements we will prove by induction:
    1. \(\hat{\delta}(q_0, w)\) contains \(q_0\) for every word \(w\text{.}\)
    2. \(\hat{\delta}(q_0, w)\) contains \(q_1\) if and only if \(w\) ends in \(0\text{.}\)
    3. \(\hat{\delta}(q_0, w)\) contains \(q_2\) if and only if \(w\) ends in \(01\text{.}\)
    Observe before continuing that only statement Item¬†3 is really required by the theorem, but to make the induction go, we need to assume something about the other states too. This is typical of inductive proofs for DFA and NFA.
    The induction begins with a base case, \(\abs{w} = 0\text{.}\) We verify the three statements. All are obvious, since \(\varepsilon\) doesn‚Äôt end in anything and \(\hat{\delta}(q_0, \varepsilon) = \{q_0\}\) by Definition¬†2.3.3.
    Next let us proceed to the inductive step. Consider a word \(w = xa\) (remember that \(x \in \aster{\Sigma}\) and \(a \in \Sigma\)). Then the length of \(w\) is one more than the length of \(x\text{.}\) In symbols, \(\abs{w} = \abs{x} + 1\text{.}\) We will assume that all three statements are true of the shorter word \(x\text{,}\) and deduce them for the longer word \(w\text{.}\)
    To see that statement 1 holds for \(w\text{,}\) we observe that by the induction hypothesis, \(q_0 \in \hat{\delta}(q_0, x)\text{.}\) Since \(w = x0\) or \(w = x1\text{,}\) and there are loop transitions from \(q_0\) labeled with both \(0\) and \(1\text{,}\) we see that \(q_0 \in \hat{\delta}(q_0, w)\text{,}\) as required.
    For statement 2, we need to prove both ‚Äúdirections‚Äù of the ‚Äúif and only if‚Äù statement. It is a bit confusing if you have not seen it before. We say ‚Äú\(P\) if and only if \(Q\)‚Äù when we wish to assert simultaneously that \(P \implies Q\) and \(Q \implies P\text{.}\) The ‚Äúforward‚Äù implication is \(P \implies Q\text{,}\) which confusingly is the ‚Äúonly if‚Äù part of the proof. The ‚Äúreverse‚Äù implication is \(Q \implies P\text{,}\) which is the ‚Äúif‚Äù part. In this case, the forward implication is
    \begin{equation*} q_1 \in \hat{\delta}(q_0, w) \implies w \text{ ends in } 0 \end{equation*}
    and the reverse implication is
    \begin{equation*} w \text{ ends in } 0 \implies q_1 \in \hat{\delta}(q_0, w)\text{.} \end{equation*}
    To prove the forward implication, we observe that only \(q_0\) has a transition to \(q_1\text{,}\) and that transition is labeled \(0\text{.}\) So the last letter of \(w\) must be \(0\text{.}\) For the reverse implication, suppose that \(w\) ends in \(0\text{.}\) Because we have assumed statement 1 for the prefix \(x\text{,}\) we know that \(\hat{\delta}(q_0, x) \ni q_0\text{.}\) Because there is a transition from \(q_0\) to \(q_1\) labeled \(0\text{,}\) it follows that
    \begin{equation*} q_1 \in \hat{\delta}(q_0, w)\text{,} \end{equation*}
    which is what we wanted to prove.
